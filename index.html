Artifact 1: Software Design and Engineering

This artifact is based on a project originally created in CS 340: Advanced Programming Concepts. The application was designed to manage and query animal shelter data using a MongoDB backend and a Python-based service layer. The original version focused on core CRUD functionality and correct interaction with the database.

I selected this artifact for my ePortfolio because it demonstrates my ability to design and structure a complete software solution using modular components. The enhancement for this category focused on improving the overall design, readability, and maintainability of the application. These changes included reorganizing functions for clearer separation of responsibilities, improving naming conventions, and adding structured documentation to better support future development and collaboration.

Through this enhancement, I demonstrated software engineering skills such as modular design, abstraction, and code organization. I also applied principles of the software development lifecycle by evaluating the original design, identifying areas for improvement, and implementing changes that improve long-term usability without altering core functionality. This artifact aligns with course outcomes related to professional software design, communication, and delivering maintainable computing solutions.
